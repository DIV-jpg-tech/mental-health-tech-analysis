# Mental Health in Tech - EDA Project

## 📌 Project Overview
This project analyzes mental health trends in the tech industry using survey data. The goal is to understand how workplace factors, demographics, and organizational policies influence mental health and treatment-seeking behavior.

## 📊 Key Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- 20+ Visualizations (Univariate, Bivariate, Multivariate)
- Business Insights & Recommendations

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📁 Dataset
- Mental Health in Tech Survey Dataset (2014)

## 🔍 Key Insights
- Majority of respondents are young professionals (25–35 age group)
- Gender imbalance observed in the tech industry
- Work interference strongly influences treatment-seeking behavior
- Workplace support systems (benefits, anonymity, leave) play a crucial role

## 💡 Business Recommendations
- Promote mental health awareness programs
- Provide clear communication about benefits
- Encourage open workplace culture
- Implement flexible leave policies

## 🚀 Conclusion
Mental health in the tech industry is influenced by both personal and workplace factors. Organizations must adopt a holistic approach to support employee well-being.
